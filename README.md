# Dressed & Dared

A single-file GitHub Pages friendly PWA for a neon outfit/location/dare randomizer.

## Deploy

Upload these files to the root of your GitHub Pages repository:

- `index.html`
- `.nojekyll`
- `README.md`

Then commit and open your GitHub Pages URL.
